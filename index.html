Dissect Reverse Engineering the Web

A guided web development project for curious coders who want to take apart and rebuild the web.

💡 Concept / Overview

5-6 hours of guided web development where you pick an existing website and reverse engineer a simplified version

Dissect is where high schoolers pick an existing website (Spotify, Notion, Pinterest, Wordle, etc.) and "reverse engineer" a simplified version using HTML, CSS, and JavaScript. Through this creative process, participants learn how modern web tools work — and remix them into something all their own.

🔧
Bonus: Add your own creative twist — maybe Spotify for dogs? Notion for witches? Pinterest for weird history memes? Get weird, make it yours!

🧩 Project Goals

1

Choose a website

Pick a site to analyze (ex. Spotify, Notion, Pinterest)

2

Dissect its structure

How does it work? What are its main components?

3

Rebuild simplified version

Use HTML, CSS, and JavaScript to recreate it

4

Add your creative remix

Make it weird, useful, or just plain cool

5

Deploy your app

Use Replit, GitHub Pages, or Vercel

6

Ship it to the gallery

30-second demo and short write-up
